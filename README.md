# The Roasted Bean - Coffee Shop Website

A premium, modern coffee shop website designed for a Phagwara-based startup.

## ☕ About the Project
**The Roasted Bean** is a artisanal coffee shop located at Law Gate, Phagwara, Punjab. This website serves as a digital storefront, showcasing our premium brews, local snacks, and community stories.

## 🚀 Features
- **Modern UI/UX**: A sleek "Midnight Roast & Gold" theme.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views using Bootstrap 5.
- **Detailed Menu**: Includes Hot Coffees, Cold Brews, Artisanal Bakery, and a special **Indian Snacks** section (Samosas, Paneer Tikka, Veggie Maggi).
- **Phagwara Localized**: Custom "About Us" and "Contact" sections highlighting our Law Gate location.
- **Coffee Chronicles Blog**: Insights into coffee culture, sustainability, and recipes.

## 🛠️ Technology Stack
- HTML5
- CSS3 (Custom Design System)
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Playfair Display & Montserrat)

## 📂 Project Structure
- `index.html`: Home page with featured experiences.
- `about.html`: Our story and vision for Phagwara.
- `menu.html`: Full categorized menu with Indian flavors.
- `blog.html`: Community stories and brewing guides.
- `contact.html`: Contact details and Google Maps integration.
- `style.css`: Unified design system.
