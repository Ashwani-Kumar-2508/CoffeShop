# The Roasted Bean - Coffee Shop Website

Welcome to the official repository for **The Roasted Bean**, a premium coffee shop startup located at Law Gate, Phagwara, Punjab.

## 🌟 Overview
This project is a modern, responsive, and visually stunning website designed to showcase a artisanal coffee experience. It caters to students, locals, and coffee enthusiasts with a curated menu and a warm, inviting community vibe.

## ✨ Features
- **Premium Design**: Midnight Roast & Gold theme with elegant typography.
- **Responsive Layout**: Built with Bootstrap 5 to ensure perfect viewing on all devices.
- **Interactive Menu**: Categories for Hot Coffee, Iced Brews, European Pastries, and local **Indian Snacks**.
- **Coffee Chronicles Blog**: A dedicated space for brewing guides and stories from the farm.
- **Localized Content**: Specific details for Law Gate, Phagwara, including an integrated Google Map.
- **Clean Structure**: Lightweight codebase without unnecessary folders for maximum performance.

## 🛠️ Technology Stack
- **HTML5** (Semantic structure)
- **CSS3** (Custom design system & animations)
- **Bootstrap 5** (Responsive grid & components)
- **Bootstrap Icons** (UI iconography)
- **Google Fonts** (Playfair Display & Montserrat)
